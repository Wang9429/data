# 海油集团股权投资全生命周期风险穿透监测平台 Demo

基于风险场景、控制活动与业务流程融合的股权投资穿透监管平台（模拟数据演示版）。

## 功能概览

- **首页驾驶舱**：集团投资总体画像、全过程风险地图、重点风险 TOP10
- **投资业务全过程监测**：业务流程树 + 业务活动详情（风险场景、控制活动、监测规则）
- **风险场景管理**：国资委46号文六大一级风险场景及二级风险
- **风险预警监测**：红/黄/蓝分级预警列表
- **风险穿透分析**：业务流程、控制、指标、权责、制度、业务触点六维穿透
- **控制活动管理**：968项控制活动清单及执行状态
- **投资事项及股东权利管理**：重大事项分类统计
- **整改闭环管理**：整改流程及事项跟踪
- **投资组合分析**：按单位、产业、区域多维分析

## 核心链路

```
国资委46号文一级风险场景 → 二级风险场景 → 投资业务环节 → 控制活动 → 监测指标 → 风险预警 → 责任主体 → 制度依据 → 整改闭环
```

## 使用方式

直接用浏览器打开 `index.html` 即可，无需安装依赖或启动服务。

```bash
# 可选：使用本地 HTTP 服务
cd cnooc-risk-platform
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 交互说明

1. 左侧菜单切换各功能页面
2. 首页风险地图点击任意环节 → 跳转至投资业务全过程监测
3. 重点风险 TOP10 / 风险预警列表点击 → 进入风险穿透分析
4. 投资业务全过程监测：投后管理 → 经营情况跟踪 可查看完整业务活动示例

## 技术栈

- 纯 HTML / CSS / JavaScript
- 无第三方依赖，单页应用架构
